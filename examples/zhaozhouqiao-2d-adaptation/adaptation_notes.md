# 改编说明

## 来源

- 源文件：`/Users/zhangweizhuo/Desktop/赵州桥故事剧本.docx`
- 原稿标题：`赵州桥·八仙试桥`
- 原稿时长：`3分钟`
- 原稿类型：`神话 / 民间故事 / 人文短片`

## 本轮改编目标

把原稿改编为一支更适合 `2D 国风动画短片` 的前期生产包，重点解决三件事：

1. 让故事从“文本可读”变成“镜头可做”
2. 让视觉从“文学描述”变成“可统一的美术方案”
3. 让制片可以把剧本、美术、分镜整理成明确代办

## 保留的核心

- 鲁班为民造桥的动机
- 张果老、柴王爷试桥的传奇结构
- 日月、五岳压桥的神话奇观
- 掌印、蹄印、车沟、膝痕、圆坑等桥上仙迹
- 匠魂通天、仙凡相通、燕赵风骨三层主题

## 主动做的改编

- 把原稿改成 `旁白主导 + 角色短对白` 的表达方式，减少大段解释性对白
- 强化 `石与光` 这组视觉母题，方便 2D 动画形成统一画面语言
- 把最重要的戏剧能量集中到 `桥心惊变 -> 鲁班托桥 -> 仙迹留桥` 这一段
- 把叙事结构改成更适合分镜拆解的 `12 镜头 / 180 秒` 版本
- 在美术层明确 `2D 手绘国风`，避免后续生产漂到写实 3D 或泛仙侠风
