# Art Package

## 整体风格

- 风格方向：`2D 国风手绘动画`
- 视觉基调：`青灰石桥 + 冷青河水 + 金色仙光 + 暖色人间灯火`
- 造型语言：`线条明确、块面清晰、带少量水墨晕染边缘，不走写实渲染`
- 动画质感：`关键帧精细，群像和环境使用有限动画与循环元素`
- 世界气质：`民间传奇，不是纯神仙宫阙；仙家力量应压进人间空间里`

## 色彩与光线

- 主色：`青灰石色、黛青、河水冷蓝`
- 辅色：`土黄、布衣米白、木轮褐色`
- 点缀色：`金色仙光、少量朱砂红`
- 日景光线：`高对比但不过曝，桥体边缘清晰，河面反光克制`
- 神迹光线：`日月金光从褡裢中溢出，五岳压桥时用沉色压低环境明度`

## 角色设计

- 鲁班：
  - 年龄感：`壮年`
  - 体态：`结实、重心低、肩背有力量`
  - 面部：`方额、浓眉、眼神定，笑时也带克制`
  - 发型：`束发，少量碎发贴在额角`
  - 服装：`粗布短衫、绑腿、旧布腰带`
  - 标志：`手上石茧、石锤、被河水打湿后的沉重布料`
- 张果老：
  - 体态：`瘦长、轻巧、带点玩笑感`
  - 面部：`长眉、带笑纹、眼神精`
  - 发型与配饰：`老者发髻，布褡裢，宽檐斗笠`
  - 标志：`倒骑白驴、褡裢隐现金光`
- 柴王爷：
  - 体态：`厚重、沉稳、步伐扎实`
  - 面部：`方脸、少言、目光直`
  - 服装：`素朴布衣，层次比鲁班更整洁`
  - 标志：`老旧独轮车、车上负重感强`
- 白驴：
  - 轮廓：`短壮，不飘`
  - 表情：`稳、倔、带仙畜灵气`
- 百姓：
  - 服装统一偏 `土黄、灰蓝、褐色`
  - 轮廓简单，避免抢主角，但必须有真实劳作感

## 服化道

- 鲁班服装要能表现 `沾水前后差异`
- 张果老的斗笠、褡裢必须能成为单独识别点
- 柴王爷的独轮车轮缘、轴心、木梁需画得扎实
- 赵州桥栏板纹样可简化，但桥体拱形、桥腹、桥栏比例不可乱改
- 日月和五岳不必写实，可以做成 `神话符号化实体`，但要有重量感

## 场景与道具

- 主场景 1：`洨河两岸`
  - 关键词：`宽河、风硬、泥岸、人群、牲口、旧路`
- 主场景 2：`新成的赵州桥`
  - 关键词：`青灰石拱、石栏、长虹卧波、桥面开阔`
- 主场景 3：`云端出场空间`
  - 关键词：`不做宏大天宫，只做云气与天光裂开`
- 主场景 4：`桥下托桥`
  - 关键词：`桥腹阴影、河水冲击、石与人对抗`
- 关键道具：
  - `石锤`
  - `布褡裢`
  - `独轮车`
  - `斗笠`
  - `日月象征体`
  - `五岳负重体`

## 一致性约束

- 鲁班的脸型、发型、体态、服装架构全片不变，只允许增加 `湿、脏、受力` 状态
- 张果老必须始终带 `白驴 + 褡裢 + 斗笠` 三个识别点中的至少两个
- 柴王爷必须始终与 `独轮车` 形成绑定关系
- 赵州桥桥拱、桥栏、桥面比例全片锁定，不可每镜重设计
- 仙迹位置要固定：`掌印、蹄印、车沟、膝痕、圆坑` 在后续镜头中要能对应回看
- 严禁漂移到写实影视光效或三维游戏材质
