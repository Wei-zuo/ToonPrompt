# Zhaozhou Bridge 2D Adaptation Example

This example shows how to run an existing folklore script through a producer-led video pipeline.

Source direction:

- project type: `2D Chinese folklore animation`
- source material: `赵州桥·八仙试桥`
- workflow: `adaptation -> story package -> art package -> storyboard -> AD feedback -> producer integration`

Included files:

- `adaptation_notes.md`
- `brief.md`
- `story_package.md`
- `art_package.md`
- `storyboard.md`
- `ad_feedback.md`
- `schedule.md`
- `production_todo.md`
- `production_packet.md`
- `producer_log.md`
- `project-state.yaml`
